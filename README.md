## Overview
This repository contains the numerical optimization models and Python analysis scripts for solving the Augmented Simeoni et al. Tumor Growth Inhibition (TGI) optimal control problem. The project focuses on minimizing tumor burden using a chemotherapeutic agent (Gemcitabine) while penalizing drug toxicity, specifically implementing a clinically realistic 5-day intermittent pulsed dosing schedule.

The optimal control problem is discretized using the Direct Collocation method (Trapezoidal Rule) and solved using the IPOPT interior-point nonlinear solver via AMPL.

## Repository Structure
### 1. AMPL Optimization Models
The project evolved through several iterations to achieve the final clinical constraints:
* **`v3.2.txt` (CASE 1)**: Uses the biologically accurate `u_max = 200.0`, proper quadratic weighting, and exports the complete matrix of physical states, transit compartments, costates (Lagrange multipliers), and the optimality switching condition.

### 2. Python Analysis Scripts
* **`plot.py`**: A high-resolution visualization dashboard using `matplotlib` and `numpy`.
* Reads the `.dat` output files generated by the AMPL solver.
* Automatically generates a clean, standalone PNG image for every state variable, control input, costate, and objective function integral.
* Uses a specialized `stem` plot for the pulsed control input `u(t)` to accurately reflect discrete dosing.


* **`partial_derivative.py`**: A symbolic mathematics engine using `sympy`.
* Constructs the full system Hamiltonian programmatically.
* Calculates the exact analytical partial derivatives to derive the five co-state differential equations.
* Outputs both the raw algebraic expansions and the cleaned, substituted forms for academic reporting.
